### Сравнительный анализ значений давления в КИС и АСУ
На добывающих скважинах цеха по добыче нефти и газа (ЦДНГ) ДД могут устанавливаться параллельно с манометрами в количестве до 3х штук для измерения буферного, линейного и затрубного давления. Параметры с ДД передаются на верхний уровнь в АСУ. Для анализа на данных скважинах с установленными ДД провели сравнение значений давления из АСУ и давления в КИС, куда оператор ДНГ вручную заносил значения устьевых давлений. По каждой выбранной скважине был сделан срез за 1 месяц, где выбрали 4 дня с периодичностью в 7 дней и только дневной обход, такой выбор и периодичность обусловлена 7 дневным вахтовым методом работы для различного спектра операторов ДНГ и его максимального охвата, а также светлое время суток. Также совместно со значением давления из КИС регистрировалось время, в которое оператор ДНГ фиксировал значение. Далее по выбранной скважине в аналогичный день и время из АСУ фиксировали значение давления с ДД. Итого по каждой выбранной скважине при наличии минимум одного ДД получили 8 значений давления - 4 значения из КИС и 4 значения из АСУ. Дополнительно для сверки показаний манометров и значений в КИС третьим лицом единоразово на половине скважин из среза были сделаны фотографии манометров и проверены даты поверки манометров. Стоит отметить, что эксплуатация манометров без поверки или с истекшим сроком поверки не допускается [13, 14]. 

Всего в срез попало 408 записи по 64 скважинам с 4-х месторождений. В процессе предобработки по различным причинам (отсутствие значения на момент среза, выбросы, нулевое значение) удалили 76 записей. 
Удалил 34 записей с отсутствием значения в АСУ, 37 записей со значением 0 в КИС КПМ, 1 запись со значением 0 в АСУ., 4 записи с отклонением более 100%. Итого удалил 76 записи. 
Осталось 332 записи.

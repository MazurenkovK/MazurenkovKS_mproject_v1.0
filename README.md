При эксплуатации месторождений углеводородного сырья на добывающих и нагнетательных скважинах установлены приборы (устройства) для измерения давления на устье скважин [3]. Они предназначены для наблюдения за параметрами эксплуатации скважин, для предупреждения аварийных ситуаций и снижение рисков негативного воздействия на окружающую среду [3][4]. Периодичность и качество переданных значений с данных приборов служит для оценки эффективности мероприятий и для регулирования процесса разработки месторождений [4]. Однако, в силу различных причин (технических неисправностей, человеческого фактора и т.д.) приборы могут передавать некорректные значения давления, что приводит к неверной оценке эффективности мероприятий и может вызвать аварийные ситуации [4]. В связи с этим возникает необходимость в создании системы, способной обнаруживать избыточное давление на устье скважин с использованием доступных средств, таких как видеокамеры смартфонов.

Цель проекта: создание приложения для детекции избыточного давления при помощи видеокамеры смартфона с приемлемыми/согласованными метриками качества.
